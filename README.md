## Environment variable

[YouTube Data API Overview](https://developers.google.com/youtube/v3/getting-started)

Create `.env` file with the following contents:
```
VUE_APP_YOUTUBE_API_KEY=<Your-YouTube-API-KEY>
```


## Project setup

```
npm install
```
